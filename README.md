# Анализ временного ряда акций (AAP) и исследование объёмов торгов

## Описание проекта

В данном проекте проводится анализ временного ряда финансовых данных компании **AAP** (акции) с использованием Python.

Основные задачи проекта:

* анализ динамики цен и объёмов торгов
* визуализация временных рядов
* агрегация данных по месяцам
* исследование автокорреляции и частичной автокорреляции
* выявление временной структуры данных

---

## Используемые данные

Файл данных:

```id="2qg6k1"
AAP_data.csv
```

Структура датасета:

| Поле   | Описание          |
| ------ | ----------------- |
| date   | дата торгов       |
| open   | цена открытия     |
| high   | максимальная цена |
| low    | минимальная цена  |
| close  | цена закрытия     |

| Name   | название компании |

## Этапы анализа данных

### 1. Загрузка и подготовка данных

Данные загружаются с преобразованием даты в индекс:

```python id="v5m2nd"
df = pd.read_csv(
```

## 3. Агрегация данных по месяцам

Данные агрегируются по месяцам с расчётом среднего значения:

```python id="x8k2qp"
df_month = df.resample("M").mean()
```

Результат позволяет анализировать долгосрочные тренды.

Построена столбчатая диаграмма:

* сглаживание временного ряда

---

## 5. Анализ автокорреляции

Для исследования структуры временного ряда используются:

* автокорреляционная функция (ACF)
* частичная автокорреляционная функция (PACF)

```python id="j3q9hf"
sm.graphics.tsa.plot_acf(df['volume'], lags=25)
sm.graphics.tsa.plot_pacf(df['volume'], lags=25)
```

---

## 6. Интерпретация ACF и PACF

* ACF показывает зависимость ряда от прошлых значений
* PACF показывает чистую зависимость на каждом лаге

Эти графики используются для:

* анализа сезонности
* выявления структуры временного ряда
* подготовки к ARIMA-моделированию
